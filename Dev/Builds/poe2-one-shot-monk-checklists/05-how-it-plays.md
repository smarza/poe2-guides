# How It Plays Checklist

## Act 1

- [ ] Use basic Monk kit.
- [ ] Use Frozen Locus as survival/saving tool.
- [ ] Use Frost Bomb to reduce enemy resistance.
- [ ] Use Killing Palm to generate charges.
- [ ] Spend charges with Falling Thunder for big damage.
- [ ] Accept that early Monk is button-heavy.
- [ ] Optional fallback if you dislike early Monk buttons:
  - [ ] Use bow setup temporarily
  - [ ] Lightning Rod + Lightning Arrow
  - [ ] Or Stormcaller Arrow until Quarterstaff switch

## Act 2

- [ ] Continue basic Monk kit.
- [ ] Keep Frozen Locus for safety.
- [ ] Keep Frost Bomb for resistance reduction.
- [ ] Keep Killing Palm for charge generation.
- [ ] Keep Falling Thunder as the big spender.
- [ ] Upgrade Quarterstaff when possible.
- [ ] Prepare for shock-loop transition.

## Act 3

- [ ] Start using the main loop when ready:
  - [ ] Storm Wave to shock
  - [ ] Siphoning Strike to extract shock
  - [ ] Siphoning Strike for big damage
  - [ ] Siphoning Strike to gain Power Charge
  - [ ] Falling Thunder when needed with 3 Power Charges

- [ ] Use Rend from second weapon set
  - [ ] Use as buff/helper
  - [ ] Helps apply shock consistently

## Act 4

- [ ] Main loop
  - [ ] Storm Wave to apply shock
  - [ ] Siphoning Strike to extract shock and deal big damage
  - [ ] Repeat Siphoning Strike on shocked ground when possible
  - [ ] Falling Thunder with 3 Power Charges when burst is needed

- [ ] Buffs and support skills
  - [ ] Use Rend to improve shock consistency
  - [ ] Use Infernal Cry before Falling Thunder
  - [ ] Use Mantra of Destruction if/when it is functional and useful

## Interludes

- [ ] Gameplay audit
  - [ ] Do I have enough damage?
  - [ ] Is shock applying consistently?
  - [ ] Is Siphoning Strike generating Power Charges?
  - [ ] Is Falling Thunder ready for burst windows?
  - [ ] Is the second weapon set configured correctly?
  - [ ] Am I dying because of gear/resists rather than skill rotation?
