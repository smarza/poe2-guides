# Build Overview — ONE SHOT Monk Leveling

## Global

- [ ] Build: ONE SHOT Monk Leveling [PoE2 0.5 League Starter]
- [ ] Class: Monk
- [ ] Ascendancy: Invoker
- [ ] League/version: 0.5 RotA
- [ ] Creator: Spud the King
- [ ] Updated: May 26, 2026
- [ ] Remember: early Monk is button-heavy, but damage improves when the shock loop and Quarterstaff upgrades come online.

## Act 1

- [ ] Start with the basic Monk kit.
- [ ] Use Falling Thunder as the big damage/charge spender.
- [ ] Use Killing Palm to generate charges for Falling Thunder.
- [ ] Use Frost Bomb to reduce enemy resistance.
- [ ] Use Frozen Locus as survival/saving tool.
- [ ] Upgrade Quarterstaff around level 4 if possible.

## Act 2

- [ ] Continue the basic Monk kit.
- [ ] Prioritize getting/using Crackling Quarterstaff near the start of Act 2.
- [ ] Keep using Frost Bomb, Frozen Locus, Killing Palm, and Falling Thunder.
- [ ] Prepare for the shock loop setup.

## Act 3

- [ ] Look for Slicing Quarterstaff near the start of Act 3.
- [ ] Start transitioning into the main shock loop when available.
- [ ] Use Storm Wave to apply shock.
- [ ] Use Siphoning Strike to consume shock, deal big damage, and gain Power Charge.
- [ ] Use Falling Thunder when needed with 3 Power Charges.

## Act 4

- [ ] Keep the shock loop as the main gameplay pattern.
- [ ] Use Rend from the second weapon set as a damage/shock helper.
- [ ] Use Infernal Cry before Falling Thunder for damage buffing when available.
- [ ] Use Mantra of Destruction when it becomes functional/useful for the setup.

## Interludes

- [ ] Review gear upgrades before continuing.
- [ ] Check resistances and survivability.
- [ ] Check whether Quarterstaff is still up to date.
- [ ] Confirm support gems and attribute requirements.
